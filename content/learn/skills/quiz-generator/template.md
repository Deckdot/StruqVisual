# Quiz Template

## For each question, structure as:
- id: unique short string
- type: single_choice | multiple_choice | short_answer
- question: clear, specific question
- choices (for choice types): 3–5 options, only 1–2 correct
- correctAnswer (for short_answer): model answer text
- explanation: 2–3 sentences explaining WHY the answer is correct and what the incorrect options missed
- points: 1–5 based on difficulty

## Anti-patterns to avoid:
- "Which of the following is NOT..." questions
- Questions with only one plausible answer
- Explanation that just restates the answer without adding insight
- Questions that test vocabulary instead of understanding
